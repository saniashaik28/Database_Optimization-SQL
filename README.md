# Database Optimization & Advanced SQL Queries

## Project Overview
Advanced SQL queries using CTEs, Subqueries, 
Indexes, and Views on Telco Customer Churn data.

##  Dataset
- Source: Kaggle - Telco Customer Churn
- Records: 7043 customers
- Features: 21 columns

##  Key Findings
- Month-to-month: High Risk (42.71% churn)
- One year: Medium Risk (11.27% churn)
- Two year: Low Risk (2.83% churn)

## SQL Concepts Used
- CTE (Common Table Expressions)
- Subqueries
- Index Creation
- Views
- CASE statements

##  Tools Used
- SQL (SQLite)
- SQLiteOnline.com
